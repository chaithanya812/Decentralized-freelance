# Decentralized Freelance Platform

A blockchain-based freelance marketplace that eliminates intermediaries between clients and freelancers.

## Description

Decentralized platform connecting clients and freelancers directly via smart contracts. Features secure escrow payments, IPFS file storage, and Web3 wallet integration. Built with Solidity, React, and Ethereum blockchain.
